Mob Programming

A Whole Team Approach

What is it?

Mob programming, also know as mobbing, is a software develpment approch. Basically, a whole team with the needed skills and knowledge work together:

  • on the same time
  • at the same time
  • in the same space
  • at the same computer

"Humans are notoriously bad at multitasking"
- Jason Kerney (Full Stack Engineer)

Mob Programming provides a solution for a developer to multitask. As people's ability to perform tasks changes on a day-to-day basis, the quality of the code that is written can vary. With Mob Programming the code is written at the highest capability at the time, since there are multiple eyes on the code.
